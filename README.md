# HeadBang- AlbumShop

## 🎼 프로젝트 소개
**앨범 재고 관리 프로그램으로, 창고 관리자가 앨범 입고/출고를 통해 관리하고 
가맹점에선 본사로 앨범을 주문하고 재고를 관리합니다.** 
<br>

 - 팀원2 : 노영록 - 출고 관리
